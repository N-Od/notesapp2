Vue Notes App
A simple notes app built with Vue 3 (CDN) and Bootstrap 5 (CDN).
It features a light blue and green color theme, a soft green header bar, and rounded square boxes for both the input area and notes.

Features

Add and delete notes

Word wrap and preserved line breaks

Rounded square input and note boxes with outlines

Light blue and green color theme

Header bar with title

Fully centered and responsive layout

How to Use

Type a note in the large rounded box.

Click Add Note to save it.

Your note appears below in a rounded green-outlined box.

Click Delete to remove a note.
